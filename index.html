Puppet PostFWD

Requirements

  • concat module

Example usage

Install PostFWD

node /box/ {
  include postfwd
}

Configure PostFWD

node /box/ {
  class { 'postfwd':
    listen_address => '127.0.0.1',
    listen_port    => '10040',
  }
}

Add rules

postfwd::rule { 'Rate limit':
  policy => 'sender_domain=example.org',
  action => 'action=rate(client_address/50/3600/450 4.7.1 sorry, max 50 requests per hour)',
  order  => '02',
}
